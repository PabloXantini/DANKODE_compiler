# About
This is a repository of an compiler for new basic programming language: DANKODE. You can see more of DANKODE [here](https://github.com/PabloXantini/DANKODE_compiler/blob/main/docs/DANKODE.md). In the other hand DANKODE language is wrapped above a Integrated Development Environment: DANKIDE.

# Getting Started
You should clone this repository for check programs, the main application entrance point where inside [src](https://github.com/PabloXantini/DANKODE_compiler/tree/main/src) folder in the file [DANKIDE.java](https://github.com/PabloXantini/DANKODE_compiler/tree/main/src/DANKIDE.java), which you must execute.
